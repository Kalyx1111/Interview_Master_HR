## 🚀 Quick Start

### Windows (Recommended)

1. Extract the ZIP anywhere.

2. Double-click:

```text
START_Interview_Master_HR.bat
```

3. Browser opens automatically.

4. Upload or paste:

- Candidate CV
- Job Description
- Experience requirements

5. Generate analysis and interview materials.
